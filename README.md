# 🎬 Cinema App

Простой клон TSUM cinema с использованием React, Vite, Redux Toolkit и RTK Query.

## 🚀 Стек

- React
- Vite
- TypeScript
- Redux Toolkit
- RTK Query
- TailwindCSS
- React Router

## 📦 Установка

```bash
git clone https://github.com/твой-ник/cinema-app.git
cd cinema-app
yarn install
yarn dev
```

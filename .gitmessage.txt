# Тип:	   	 Что значит:
# feat - 		 Новая фича
# fix	-  		 Исправление бага
# refactor - Улучшение кода без изменения функционала
# style	- 	 Только правки в стиле (пробелы, переносы, prettier)
# chore	- 	 Обслуживание (обновление пакетов, конфиги и т.д.)
# docs - 	   Добавление или изменение тестов
#
# Формат: <тип>: краткое описание изменений (до 72 символов)
#
# Примеры:
# feat: добавила компонент карточки фильма
# fix: исправила баг с отображением постера
# refactor: вынесла api-запросы в отдельный модуль
